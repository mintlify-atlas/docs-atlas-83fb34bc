# Welcome to XW CLI

XW CLI is a command-line tool that enables zero-configuration deployment of large language models on Chinese domestic AI chips, including Huawei Ascend NPUs and other hardware platforms. It eliminates complex environment setup and provides a simple, one-command workflow to run production-grade AI inference.

## How It Works

XW CLI provides a unified interface to manage the complete lifecycle of AI model deployment:

<Steps>
  <Step title="Start the Server">
    Launch the XW HTTP server to manage model instances
    
    ```bash
    xw serve
    ```
  </Step>
  
  <Step title="Pull a Model">
    Download pre-configured models from the registry
    
    ```bash
    xw pull qwen3-8b
    ```
  </Step>
  
  <Step title="Run the Model">
    Start inference with a single command
    
    ```bash
    xw run qwen3-8b
    ```
  </Step>
</Steps>

## Supported Hardware

XW CLI supports a wide range of domestic AI accelerators:

- **Huawei Ascend Series**: 910B, 910B1, 910B4, 310P
- **MetaX Series**: C550
- Extensible architecture for additional hardware platforms

## Supported Models

XW CLI comes with pre-configured support for popular Chinese and international LLMs:

- **Qwen Series**: Qwen2.5, Qwen3 (7B, 8B, 32B, 235B variants)
- **GLM Series**: GLM-4.7, GLM-OCR
- **MiniMax**: MiniMax-2.1, MiniMax-2.5
- **DeepSeek**: DeepSeek-OCR
- Custom model support through configuration

## Architecture

XW CLI consists of three main components:

1. **CLI Client**: User-facing command-line interface
2. **HTTP Server**: Model management and API gateway
3. **Runtime Engines**: Multiple inference backends (vLLM, MindIE, MLGuider, Omni-Infer)

The server automatically detects available hardware, selects the optimal inference engine, and manages Docker-based runtime isolation for each model instance.
